# 📋 Resumo Executivo - Rota de Ataque Questões

## 🎯 **Projeto**
Plataforma online para estudo de concursos públicos com modelo freemium

## 🛠️ **Stack Tecnológica**
- **Backend**: Node.js + Express + MySQL + Sequelize
- **Frontend**: React 18 + TypeScript
- **Autenticação**: JWT
- **Pagamentos**: API Asaas
